# Hero Images

Please add the following hotel images to this directory:

1. `hotel-exterior-1.jpg` - Hotel front entrance with modern architecture
2. `hotel-exterior-2.jpg` - Hotel exterior with parking area
3. `hotel-exterior-3.jpg` - Hotel side view showing room windows
4. `hotel-pool.jpg` - Hotel outdoor dining area with pool

These images will be displayed in the hero slider on the homepage, automatically transitioning every 5 seconds.

## Image Requirements:
- Format: JPG or PNG
- Recommended size: 1920x1080px (Full HD)
- Aspect ratio: 16:9
- File size: Optimized for web (under 500KB each)

## Current Slider Features:
- Auto-advance every 5 seconds
- Smooth fade transitions
- Clickable indicators at the bottom
- Responsive design for mobile/tablet/desktop
